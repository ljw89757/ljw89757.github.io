删除仓库中的所有文件，但保留 Git 历史记录

这种方法会将所有文件从当前分支中删除，但保留 Git 的提交历史记录。适用于希望清理项目文件但保留版本历史的情况。

1. 克隆或进入现有仓库
2. 删除所有文件和文件夹
3. 提交更改
4. 推送到远程仓库
